Snowpack & npm

We use snowpack in all of our generated static site starters. It has allowed us to ensure the same build process happens across them, and we can introduce Glitch specific plugins in the future.

Snowpack has kept on the edge of what browsers can support natively, and provides ES modules directly from package. We aren't using that in this starter, but might in the future.

  • Try this: Want to see how Glitch helps you use and update new Node packages? Edit the package.json file in your project and you'll see an "Add Package" button that helps keep your project up to date.
